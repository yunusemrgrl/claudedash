Quick Start

Up in 30 seconds.

Terminal 
# Start watching default Claude tasks folder
$ npx agent-scope start

# Custom project path
$ npx agent-scope start --claude-dir ~/my-project/.claude

# Custom port
$ npx agent-scope start --port 3000

01

Run the CLI

Fastify server starts at localhost:4317. Chokidar watches ~/.claude/tasks/ and .agent-scope/ for changes.

02

Agent writes tasks

Claude Code's TodoWrite tool writes task files. agent-scope picks them up instantly and broadcasts a sessions SSE event.

03

Dashboard updates live

No polling. No refresh needed. SSE pushes changes to all open browser tabs within ~100ms of every file write.

Git Worktrees

Multi-repo agent visibility.

Running agents across multiple git worktrees? agent-scope maps sessions to branches by cwd and shows dirty state, ahead/behind counts, and associated tasks per worktree.
